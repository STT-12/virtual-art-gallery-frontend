#  虚拟艺术画廊 - 前端

一个沉浸式、交互式的在线艺术展览平台，支持2D/3D画廊浏览、个人画廊定制、社区互动等功能。本项目为课程设计全栈项目的前端部分，使用 Vue 3 技术栈开发。

---

##  项目预览

> 详细设计文档请见 [docs/虚拟艺术画廊系统说明文档.pdf](./docs/虚拟艺术画廊系统说明文档.pdf)。

##  技术栈

- **核心框架**：Vue 3 (Options API + `<script setup>`)
- **路由管理**：Vue Router 4
- **状态管理**：Vuex 4
- **构建工具**：Vite
- **3D 渲染**：Three.js (用于3D行星画廊)
- **2D 渲染**：Canvas API (用于动态星空/网格背景)
- **样式**：CSS3 (毛玻璃、响应式布局)

---

##  主要功能模块

### 1. 画廊浏览与探索
- **2D漫游画廊**：通过鼠标拖拽平移、滚轮缩放，在宇宙空间中自由探索瀑布流分布的艺术作品。支持风格筛选和艺术时间轴快速定位。
- **3D行星画廊（加分项）**：基于 Three.js，艺术作品环绕行星陈列，用户可旋转、缩放视角，点击时间轴定位相关作品。

### 2. 画作深度展示
- **画作详情页**：展示高清大图、标题、作者、年代、风格、描述及背景故事。
- **交互式放大镜**：鼠标悬停可局部放大，支持滑块调节缩放倍数。
- **背景音乐播放器**：每幅画作配有专属音乐，支持播放/暂停、音量调节、进度拖拽。

### 3. 个人画廊定制
- **创建与编辑**：注册用户可创建个人画廊，自由命名、添加描述、选择主题背景。
- **自由布局**：通过拖拽将作品放入工作区，可调整位置、大小、旋转角度，并保存布局。
- **权限设置**：支持公开、好友可见、私密三种访问权限。

### 4. 社区与互动
- **社交系统**：搜索添加好友、处理好友请求、实时聊天。
- **画廊分享**：可将自己的画廊或喜爱的作品分享给好友。
- **点赞评论**：对画作进行点赞、发表评论。

### 5. 用户认证
- 完整的注册/登录功能，基于 JWT 身份验证，路由守卫保护需要登录的页面。
